Програма для аналізу вартості каналів зв'язку філії.

Дозволяє аналізувати вартість каналів зв'язку відділень на підставі даних сайту: https://rp.privatbank.ua/rp_v3/

Дані можуть вибиратися за заданий період і групуватися по відділенням або провайдерам. Є можливість порівняння по датах періоду. Якщо дати періоду не задані, передбачається поточна дата. Для сортування клікніть по відповідній назві колонці в шапці таблиці

Дані для аналізу беруться з сайту:https://rp.privatbank.ua/rp_v3/ Для цього треба вивантажити з цього сайту дані по відділеннях в XLS-файл, причому з дочірніми і закритими каналами зв'язку. Потім в меню [Адміністрування] необхідно завантажиті цей файл в програму. Для цього у Вас повинен бути доступ адміністратора. При зміни даних на сайті, процедуру треба повторити.

В розрахунок беруться тількі канали з не нульовою вартістю.

Стек технологій:

  • Angular2
  • NodeJS
  • WebPack
  • Tomcat
  • Spring
  • Hibernate
