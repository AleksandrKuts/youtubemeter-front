Сервис статистики каналов Youtube

Написан под влиянием проекта: dislikemeter.com

Аналитика Youtube канала: накрутка лайков, накрутка дизлайков, скручивание просмотров и т.д.

Служит для анализа работы каналов youtube, отслеживает статистику по видео: лайки, дизлайки, просмотры, комментарии. Это позволяет зафиксировать аномальное изменение этих характеристик, например:

  • Скручивание просмотров видео: BOEHHO-БACKETБOЛЬHЫE COЛДATЫ
  • Детализация скручивания просмотров видео: BOEHHO-БACKETБOЛЬHЫE COЛДATЫ
  • Накручивание дизлайков видео: В Кремле паника: Анаконда на границе с РФ. У Украины теперь есть ударные БПЛА.
  • Накрутка дизлайков видео: Е. Понасенков: тайна Возрождения, задания в посольстве, трагедия в Керчи, Вермахт и 1941 год

Работа сервиса основана на плейлистах, список которых ведет администратор - он же автор программы. Сбор метрик ведется только для видео из этого списка. Собираются следующие метрики:

  • лайки
  • дизлайки
  • просмотры
  • комментарии

Для сбора метрик используется сервис: Youtube API

Период за который собираются метрики - две недели с момента опубликования видео, дольше собирать метрики не имеет смысла. Метрики собираются раз в минуту.

Сервис периодически, раз в три минуты, проверяет плейлисты на появление новых видео, и автоматически добавляет их в обработку.

Сбор метрик начинается с момента добавления администратором плейлиста, по-этому данные метрик по старым видео могут отсутствовать или быть неполными (только с даты добавления).

Список плейлистов не полностью отражает предпочтения автора, некоторые добавлены как хороший объект для исследования. Также в списке присутствуют нейтральные не политические каналы в качестве контрольной группы.

Сервис написан с целью объективного исследования влияния ботов на ютуб, прежде всего для понимания этого явления самим автором.

Обращаю внимание что метрики выдаваемые через api могут отличаться от тех что отображены под видео на сайте Youtube . Пока я наблюдаю это для дизлайков, наверное как следствие их накручивания.

Например видео Е. Понасенков: тайна Возрождения, задания в посольстве, трагедия в Керчи, Вермахт и 1941 год., на 2018.11.10 17:21 под видео 132 дизлайка, через api 516. Это совпадает с накруткой дизлайков в период с 19:38 по 19:45 8.11.2018г., с этого момента количество метрик различается.
