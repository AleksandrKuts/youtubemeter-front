Сервіс статистики каналів Youtube

Написаний під впливом проекту: dislikemeter.com

Аналітика Youtube каналу: накрутка лайків, накрутка дізлайков, скручування переглядів тощо

Служить для аналізу роботи каналів youtube, відстежує статистику по відео: лайки, дизлайки, перегляди, коментарі. Це дозволяє зафіксувати аномальну зміну цих характеристик, наприклад:

  • Скручування переглядів відео: BOEHHO-БACKETБOЛЬHЫE COЛДATЫ
  • Деталізація скручування переглядів відео: BOEHHO-БACKETБOЛЬHЫE COЛДATЫ
  • Накручування дизлайків відео: В Кремле паника: Анаконда на границе с РФ. У Украины теперь есть ударные БПЛА.
  • Накрутка дизлайків відео: Е. Понасенков: тайна Возрождения, задания в посольстве, трагедия в Керчи, Вермахт и 1941 год

Робота сервісу заснована на плейлистах, список яких веде адміністратор - він же автор програми. Збір метрик ведеться тільки для відео з цього списку. Збираються такі метрики:

  • лайки
  • дизлайки
  • перегляди
  • коментарі

Для збору метрик використовується сервіс: Youtube API

Період за який збираються метрики - два тижні з моменту опублікування відео, довше збирати метрики не має сенсу. Метрики збираються раз на хвилину.

Сервіс періодично, раз в три хвилини, перевіряє плейлисти на появу нових відео, і автоматично додає їх в обробку.

Збір метрик починається з моменту додавання адміністратором плейлиста, по-цьому дані метрик по старим відео можуть бути відсутніми або бути неповними (тільки з дати додавання).

Список плейлистів не в повному обсязі відбиває вподобання автора, деякі додані як хороший об'єкт для дослідження. Також в списку присутні нейтральні не політичні канали в якості контрольної групи.

Сервіс написаний з метою об'єктивного дослідження впливу ботів на ютуб, перш за все для розуміння цього явища самим автором.

Звертаю увагу що метрики які видаються через api можуть відрізнятися від тих що відображені під відео на сайті Youtube. Поки я спостерігаю це для дизлайків, напевно як наслідок їх накручування.

Наприклад відео: Е. Понасенков: тайна Возрождения, задания в посольстве, трагедия в Керчи, Вермахт и 1941 год., на 2018.11.10 17:21 під відео 132 дизлайка, по api 516. Це співпадає з накручуванням дизлайків за період з 19:38 по 19:45 8.11.2018г., з цього моменту кількість метрик різниться.

Сайт використовує такі cookies:

  • langinterface - Мова інтерфейсу: 'en', 'ua', 'ru' (за замовчуванням - 'ua')
  • playlistid - останній переглянутий плейлист, при наступному заході на сайт перегляд почнеться з нього.
